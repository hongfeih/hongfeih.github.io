    AdaptivePlayer

    Extends:

    FakeEventTarget → AdaptivePlayer

    Implements:

    • *

    Endeavor Streaming HTML5 Adaptive Player
    Support DASH with Shaka player, support HLS with native player and hls.js
    And HTTP video (mp4/ogg/webm) with native support

    Example:

    let player = new AdaptivePlayer(document.getElementById('video'));

    NameTypeAttributeDescription
    config object

    player configurations
    config.withCredentials boolean

    false by default
    config.withCredentialsFragmentPattern string[]

    set to use withCredentials for segment url with defined values, by default ['/nlds/', '/nldsu/', '/nlds_vod/', '/nldsu_vod/']

    Use external m3u8 parser for native player (DRM streams on Mac and play on iOS browsers) to get UTC time and AD-stitching related tags

    public load(url: string, startTime: number, streamType: StreamType) source

    Play video with url, Return error when url is wrong, and current playback will not be stopped while playing

    Params:

    NameTypeAttributeDescription
    url string

    playback url
    startTime number
    • optional

    start playback from certain position in second;
    if not set or set as -1, will start from 0 (VOD) or live edge (Live)
    if set as 0, will start from 0 (VOD) or first segment in list (Live)
    For HLS Live, time may be not accurate as player will seek to start time of TS chunk, i.e., pass 1501768500 will go to 1501768510
    streamType StreamType
    • optional

    to set streaming type in case of url doesn't end with explicit extension; auto set according to url if set as null
