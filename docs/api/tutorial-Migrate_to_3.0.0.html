Migrate to 3.0.0

1. DASH tracks change

shaka-player uses variants (mix audio tracks and video tracks) instead of tracks from 2.3.x.
We still use videoTracks/audioTracks/textTracks, but need to modify that part how you use it.
Monitoring TrackChange event to catch up variant id changing:

if (event && event.details) {
    if (event.details.type == AdaptivePlayer.Kind.AUDIO) {
        // update audio tracks
        var audioTracks = event.details.tracks;
        if (audioTracks) {
            ...
        }
    } else if (event.details.type == AdaptivePlayer.Kind.VIDEO) {
        // update audio tracks
        ...
    } else if (event.details.type == AdaptivePlayer.Kind.TEXT) {
        // update text tracks
        ...
    }
}

Please refer to tests/tests_tracks.html for details.

2. PlayEnd

Player will send PlayEnd event on completion (StateManager.ActionReason.COMPLETE) or user abort (StateManager.ActionReason.ABORT)

player.addEventListener(AdaptivePlayer.EventType.PlayEnd, function (evt) {
    console.log('PlayEnd reason = ' + evt.details.reason);
});

3. AutoPlayStatus

Player will send AutoPlayStatus event when auto play called on chrome.

player.addEventListener(AdaptivePlayer.EventType.AutoPlayStatus, function (evt) {
  if (!e.details.autoplay) {
    // autoplay disabled, show play button
    // ...
  }
});
