Tutorial: Basic usage

Basic usage

Files

javascript files need to be included into the page.

<script src="nladaptiveplayer.js"></script>

Play a stream

1. Create a player instance.

var video = document.getElementById('video_main');
var player = new AdaptivePlayer(video);

2. Set debug log level

player.setLogLevel(AdaptivePlayer.LogLevel.DEBUG); // set debug log level

3. Set configuration

Please refer to Configurations

4. Register events

Register events needed for playback.
Please refer to Events how-to

5. Load the manifest.

var startTime = 30; //in seconds
var protocol = AdaptivePlayer.Protocol.DASH; //AdaptivePlayer.Protocol.HLS or AdaptivePlayer.Protocol.HTTP for mp4
player.load(manifest_url, startTime, protocol); //utc time for live stream
